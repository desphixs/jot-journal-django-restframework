# Build Jot: A Developer Journal API

### Project Description

You are going to build Jot, a developer-focused journaling API using Django and Django REST Framework. The twist is that each journal entry can have multiple tags attached to it — things like #bugfix, #learning, or #deepwork. This means you will be working with a Many-to-Many relationship for the first time. You will build the models, wire up the serializer to handle the tags correctly, and create clean API endpoints so a frontend app can create, read, update, and delete both entries and tags. By the end, you will understand one of the most important concepts in backend development — how to link data together in a database and serve it cleanly over an API.

## Tech Stack

- Django
- Django REST Framework
